Practical Exercice - The treasure map
=======================

## Install

To install the project, you need yarn :
```
npm i -g yarn
```

Then you can install the dependencies
```
yarn
```

## Build And Run

To build the project, use
```
yarn build
```

Then you can run the project with :
```
node .
```

Or :
```
./index.js
```

## Test

To run the test suite, use :
```
yarn test
```
